# YT-DLP 下载器 v1.0.0

## 更新内容

首次发布，包含以下功能：
- 图形界面操作，简单易用
- 支持输入视频URL进行下载
- 提供多种视频质量选项（最佳质量、1080p、720p、480p）
- 自动使用YouTube-Cookie.txt进行授权
- 可自定义下载保存位置
- 实时显示下载进度

## 使用说明

1. 下载并解压发行版压缩包
2. 确保以下文件在同一目录下：
   - yt_dlp_gui.exe（主程序）
   - yt-dlp.exe（下载核心）
   - ffmpeg.exe（视频处理工具）
3. 运行yt_dlp_gui.exe启动程序
4. 在程序界面输入YouTube视频URL
5. 选择视频质量和保存位置
6. 点击下载按钮开始下载

## 注意事项

- 如需下载需要登录的视频，请将YouTube-Cookie.txt文件放在程序同目录下
- 下载高质量视频时会自动合并视频和音频流，请耐心等待
- 建议使用稳定的网络连接以确保下载顺利完成
